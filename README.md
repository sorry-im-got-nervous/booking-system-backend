# Booking System Backend

## Описание проекта

Система предназначена для управления бронированием общих ресурсов организации: конференц-залов, оборудования, транспортных средств и других ресурсов.

Проект реализует серверную часть приложения и предоставляет REST API для работы с пользователями, ресурсами и бронированиями.

## Технологический стек

* Node.js
* Express.js
* MySQL
* JWT Authentication
* bcrypt
* REST API

## Основные возможности

* регистрация и авторизация пользователей;
* управление ресурсами;
* создание и отмена бронирований;
* предотвращение двойного бронирования;
* система уведомлений;
* разграничение прав доступа.

## Структура проекта

* users
* resources
* bookings
* notifications

## Запуск проекта

Установка зависимостей:
```powershell
npm install
```

Запуск в режиме разработки:
```powershell
npm run dev
```

Запуск в рабочем режиме:
```powershell
npm start
```
